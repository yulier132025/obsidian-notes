现实：好累→→→→→→→


```
逻辑：拖延，完美主义压力，第一次复盘没思路

明显的循环：
 时间与完美主义压力 → 没有思路 → 无法推进 → 用转移缓解 → 无效但没有其他方法 → 强迫回归 → 更卡 → 再逃避
 
 
```
